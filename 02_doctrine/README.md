_(현재 등록된 doctrine 없음)_

DC-001~003은 axiom으로 승급 완료.
DC-004~008은 승급 조건(3개+ 프로젝트 검증) 미충족으로 praxis로 강등됨. (2026-03-17)

- 위반 시 실제 피해가 발생한 사례가 있을 것
